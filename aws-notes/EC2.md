Amazon EC2

What is EC2?

Amazon EC2 (Elastic Compute Cloud) is a virtual server service provided by AWS.

Uses of EC2

- Hosting websites
- Running applications
- Testing software
- Hosting APIs

Key Concepts

Instance

A virtual machine running in AWS.

Security Group

Acts as a firewall for EC2.

SSH

Used to securely connect to Linux servers.

Default Port: 22

Common Ports

- HTTP : 80
- HTTPS : 443
- SSH : 22

 Example :-

A company hosts its website on an EC2 instance and allows users to access it through the internet.
